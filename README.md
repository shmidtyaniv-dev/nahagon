# מרכז הרכב של מור בית השקעות

מערכת ניהול צי רכב ו-CRM עבור מור בית השקעות.

## קובץ ראשי

- **[fleetos_114.html](./fleetos_114.html)** - מערכת המלאה (single-page HTML)

## פריסה

המערכת פרוסה אוטומטית ל-Vercel בכל push ל-`main`:
- https://nahagon.vercel.app/
- https://nahagon.vercel.app/fleetos_114.html

## טכנולוגיות

- HTML / CSS / JavaScript (ללא framework)
- Firebase Firestore (סנכרון נתונים)
- PDF.js + pdf-lib (הנדסת PDF)
- SheetJS (ייצוא Excel)
- Anthropic Claude API (סריקת רישיונות AI)

## פיתוח

כל שינוי ב-`fleetos_114.html` מתפרס אוטומטית לפרודקשן.
